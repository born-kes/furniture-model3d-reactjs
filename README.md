## Get Started With the Code

---

Install node.js

```PowerShell
    $ cd <furnite-model3d-reactjs folder>
    $ npm install -f
    $ npm run build
    $ npm start
```

## Get Started Test

```PowerShell
    $ cd <furnite-model3d-reactjs folder>
    $ npm run test
```
